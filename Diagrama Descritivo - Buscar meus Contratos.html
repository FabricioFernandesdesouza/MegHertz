Buscar meus Contratos

Caso de Uso EC004 - Buscar meus contratos
Ator Primário Locatário de veículos automotivos.
Ator Secundário IA.
Visão Geral Este caso de uso descreve a busca de um contrato de aluguel.
Atributos numero da apólice.
Pré-condição Falar no chat que quer buscar um contrato.
Fluxo Principal A Meg perguntará ao usuário o numero da apólice que ele quer visualizar.
Usuário informa o numero da apólice.
É buscado por API o pdf do contrato informado pelo usuário.
O pdf do contrato é exibido pro usuário.
Fluxo Alternativo
Fluxo de Exceção O usuário informa uma apólice não existente.
A Meg explica que o numero da Apólice está incorreto ou não existe.
Pós Condições O contrato é disponibilizado para o usuário.
