Buscar Agencia Hertz

Caso de Uso EC003 - Buscar Agencia Hertz
Ator Primário Locatário de veículos automotivos.
Ator Secundário IA.
Visão Geral Este caso de uso descreve a buscar de uma agencia Hertz.
Atributos Endereço
Pré-condição Falar no chat que quer busca uma agencia.
Fluxo Principal O usuário irá digitar um endereço.
A IA ira usar o endereço digitado pelo usuário barra buscar suas coordenadas.
As coordenadas serão comparadas com o uma lista de endereços de agencia Hertz.
A agencia com o endereço mais próximo ao informado pelo usuário será exibido no chat.
Fluxo Alternativo Não tem.
Fluxo de Exceção O usuário informa um endereço inexistente.
Pós Condições
