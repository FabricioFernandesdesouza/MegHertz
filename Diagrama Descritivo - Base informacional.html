Base informacional

Caso de Uso EC002 - Consultar base informacional
Ator Primário Locatário de veículos automotivos.
Ator Secundário IA.
Visão Geral Este caso de uso descreve o acesso a base informacional.
Atributos Input do Ator Primário e Input do Ator Secundário.
Pré-condição Abrir o chat com a Meg.
Fluxo Principal Usuário digita uma duvida que ele tem.
A IA identifica a intenção do usuário.
IA busca a resposta adequada.
IA digita a resposta no chat.
Fluxo Alternativo A IA não identifica uma intenção ou resposta adequada.
Explica para o usuário que não entendeu e pede para escrever novamente ou de outra maneira.
A IA identifica a intenção do usuário.
Repete os dois últimos passos do fluxo principal.
Fluxo de Exceção A IA não identifica uma intenção ou resposta adequada duas vezes.
A IA explica ao usuário que no momento ela não tem uma resposta para está pergunta e indica um numero de telefone para contato.
Pós Condições A duvida do usuário é sanada.
